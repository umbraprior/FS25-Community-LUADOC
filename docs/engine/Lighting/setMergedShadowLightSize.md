### setMergedShadowLightSize

**Description**

> Sets the soft shadow light size for the merged shadow of the given light source (if it belongs to any merged shadow).

**Definition**

> setMergedShadowLightSize(entityId lightId, float lightSize)

**Arguments**

| entityId | lightId   | id of the light source node                 |
|----------|-----------|---------------------------------------------|
| float    | lightSize | soft shadow light size of the merged shadow |
