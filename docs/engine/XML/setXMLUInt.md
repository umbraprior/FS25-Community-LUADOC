### setXMLUInt

**Description**

> Set XML file unsigned integer value (in the range 0 - 4'294'967'295). If provided with a decimal number any decimals
> will be rounded in the saved value.

**Definition**

> setXMLUInt(entityId xmlId, string nodePath, integer value)

**Arguments**

| entityId | xmlId    | xmlId                            |
|----------|----------|----------------------------------|
| string   | nodePath | path to XML element or attribute |
| integer  | value    | value                            |
