### streamWriteIntN

**Description**

> Stream write N bit signed integer. 0 < N < 32.

**Definition**

> streamWriteIntN(entityId streamId, integer value, integer numberOfBits)

**Arguments**

| entityId | streamId     | streamId     |
|----------|--------------|--------------|
| integer  | value        | value        |
| integer  | numberOfBits | numberOfBits |
