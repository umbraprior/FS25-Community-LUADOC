> Get height of the density map at the world position

**Definition**

> getDensityHeightAtWorldPos(entityId fillDataPlaneId, float x, float y, float z)

**Arguments**

| entityId | fillDataPlaneId | fillDataPlaneId |
|----------|-----------------|-----------------|
| float    | x               | x               |
| float    | y               | y               |
| float    | z               | z               |

**Return Values**

| float | height |                   |
|-------|--------|-------------------|
| float | delta  | height to terrain |
