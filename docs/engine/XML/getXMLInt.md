### getXMLInt

**Description**

> Get XML file integer value. If used on a decimal number any decimals will be cut off.

**Definition**

> getXMLInt(entityId xmlId, string nodePath)

**Arguments**

| entityId | xmlId    | xmlId                            |
|----------|----------|----------------------------------|
| string   | nodePath | path to XML element or attribute |

**Return Values**

| integer | value | value |
|---------|-------|-------|
