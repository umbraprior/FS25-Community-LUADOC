### setTranslation

**Description**

> Set translation of a transform object in its local space (relative to parent node)

**Definition**

> setTranslation(entityId transformId, float x, float y, float z)

**Arguments**

| entityId | transformId | transformId                           |
|----------|-------------|---------------------------------------|
| float    | x           | x translation relative to parent node |
| float    | y           | y translation relative to parent node |
| float    | z           | z translation relative to parent node |
