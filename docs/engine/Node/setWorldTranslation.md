### setWorldTranslation

**Description**

> Set translation of a transform object in world space

**Definition**

> setWorldTranslation(entityId transformId, float wx, float wy, float wz)

**Arguments**

| entityId | transformId | transformId   |
|----------|-------------|---------------|
| float    | wx          | x translation |
| float    | wy          | y translation |
| float    | wz          | z translation |
