### drawDebugLine

**Description**

> Render a line. Only use for debug rendering

**Definition**

> drawDebugLine(float x0, float y0, float z0, float r0, float g0, float b0, float x1, float y1, float z1, float r1,
> float g1, float b1, boolean? solid)

**Arguments**

| float    | x0    | x0                                                                                                                          |
|----------|-------|-----------------------------------------------------------------------------------------------------------------------------|
| float    | y0    | y0                                                                                                                          |
| float    | z0    | z0                                                                                                                          |
| float    | r0    | r0                                                                                                                          |
| float    | g0    | g0                                                                                                                          |
| float    | b0    | b0                                                                                                                          |
| float    | x1    | x1                                                                                                                          |
| float    | y1    | y1                                                                                                                          |
| float    | z1    | z1                                                                                                                          |
| float    | r1    | r1                                                                                                                          |
| float    | g1    | g1                                                                                                                          |
| float    | b1    | b1                                                                                                                          |
| boolean? | solid | [optional] true: depth is checked and point can be hidden behind other meshes; false: always rendered on top (default=true) |
