### printWarning

**Description**

> Print given values to console
> Each argument is printed in a separate line
> Can handle all data types without explicit casting
> Text is colored red in the script console and triggers automatic opening of console if "openOnError" setting is
> enabled

**Definition**

> printWarning(any? arg1, any? arg2, any? arg3, any? arg4, any? arg5, any? arg6, any? arg7, any? arg8)

**Arguments**

| any? | arg1 | arg1 [optional] |
|------|------|-----------------|
| any? | arg2 | arg2 [optional] |
| any? | arg3 | arg3 [optional] |
| any? | arg4 | arg4 [optional] |
| any? | arg5 | arg5 [optional] |
| any? | arg6 | arg6 [optional] |
| any? | arg7 | arg7 [optional] |
| any? | arg8 | arg8 [optional] |
