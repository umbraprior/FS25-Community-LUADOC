### localDirectionToLocal

**Description**

> Local space to local space transformation, only direction without translation

**Definition**

> localDirectionToLocal(entityId transformId, entityId targetTransformId, float dx, float dy, float dz)

**Arguments**

| entityId | transformId       | transformId       |
|----------|-------------------|-------------------|
| entityId | targetTransformId | targetTransformId |
| float    | dx                | dx                |
| float    | dy                | dy                |
| float    | dz                | dz                |

**Return Values**

| float | dx | dx |
|-------|----|----|
| float | dy | dy |
| float | dz | dz |
