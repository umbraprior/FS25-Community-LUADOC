### project

**Description**

> Transform vector from world space into screen space

**Definition**

> project(float wx, float wy, float wz)

**Arguments**

| float | wx | wx |
|-------|----|----|
| float | wy | wy |
| float | wz | wz |

**Return Values**

| float | sx | sx |
|-------|----|----|
| float | sy | sy |
| float | sz | sz |
