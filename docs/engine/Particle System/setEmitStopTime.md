### setEmitStopTime

**Description**

> Set emitter stop time.
> The particleSystemId can be retrieved by using getGeometry() on the shape/node

**Definition**

> setEmitStopTime(entityId particleSystemId, float emitStopTime)

**Arguments**

| entityId | particleSystemId | particleSystemId |
|----------|------------------|------------------|
| float    | emitStopTime     | emitStopTime     |
