### unProject

**Description**

> Transform vector from screen space into world space

**Definition**

> unProject(float sx, float sy, float sz)

**Arguments**

| float | sx | screenspace x [0..1]         |
|-------|----|------------------------------|
| float | sy | screenspace y [0..1]         |
| float | sz | screenspace z / depth [0..1] |

**Return Values**

| float | wx | wx |
|-------|----|----|
| float | wy | wy |
| float | wz | wz |
