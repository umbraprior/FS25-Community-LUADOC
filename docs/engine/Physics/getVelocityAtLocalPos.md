**Definition**

> getVelocityAtLocalPos(entityId transformId, float positionX, float positionY, float positionZ)

| entityId | transformId | transformId |
|----------|-------------|-------------|
| float    | positionX   | positionX   |
| float    | positionY   | positionY   |
| float    | positionZ   | positionZ   |

**Return Values**

| float | velocityX | velocityX |
|-------|-----------|-----------|
| float | velocityY | velocityY |
| float | velocityZ | velocityZ |
