**Definition**

> localToLocal(entityId transformId, entityId targetTransformId, float x, float y, float z)

| entityId | transformId       | transformId       |
|----------|-------------------|-------------------|
| entityId | targetTransformId | targetTransformId |

| float    | y                 | y                 |
| float    | z                 | z                 |

**Return Values**

| float | x | x |
|-------|---|---|
| float | y | y |
| float | z | z |
