### setOverlayUVs

**Description**

> Set overlay uv coordinates

**Definition**

> setOverlayUVs(entityId overlayId, float u0, float v0, float u1, float v1, float u2, float v2, float u3, float v3)

**Arguments**

| entityId | overlayId | overlayId                               |
|----------|-----------|-----------------------------------------|
| float    | u0        | bottom left u tex coord (original = 0)  |
| float    | v0        | bottom left v tex coord (original = 0)  |
| float    | u1        | top left u tex coord (original = 0)     |
| float    | v1        | top left v tex coord (original = 1)     |
| float    | u2        | bottom right u tex coord (original = 1) |
| float    | v2        | bottom right v tex coord (original = 0) |
| float    | u3        | top right u tex coord (original = 1)    |
| float    | v3        | top right v tex coord (original = 1)    |
