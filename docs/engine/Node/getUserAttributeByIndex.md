**Definition**

> getUserAttributeByIndex(entityId objectId, integer attributeIndex)

**Arguments**

| entityId | objectId       | objectId                     |

| integer  | attributeIndex | attributeIndex starting at 0 |

**Return Values**

| any?    | value | [optional]                      |
|---------|-------|---------------------------------|
| string  | name  |                                 |
| integer | type  | Value of UserAttributeType enum |
