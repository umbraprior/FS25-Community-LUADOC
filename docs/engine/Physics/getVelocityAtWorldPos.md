### getVelocityAtWorldPos

**Description**

> Get velocity at world position of transform object (only for dynamic physics objects)

**Definition**

> getVelocityAtWorldPos(entityId transformId, float positionX, float positionY, float positionZ)

**Arguments**

| entityId | transformId | transformId |
|----------|-------------|-------------|
| float    | positionX   | positionX   |
| float    | positionY   | positionY   |
| float    | positionZ   | positionZ   |

**Return Values**

| float | velocityX | velocityX |
|-------|-----------|-----------|
| float | velocityY | velocityY |
| float | velocityZ | velocityZ |
