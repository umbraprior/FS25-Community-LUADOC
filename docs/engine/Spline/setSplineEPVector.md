### setSplineEPVector

**Description**

> Set the position of a spline edit point.

**Definition**

> setSplineEPVector(integer shapeId, integer index, vector position)

**Arguments**

| integer | shapeId  |                                                             |
|---------|----------|-------------------------------------------------------------|
| integer | index    | The index of the edit point to be positioned, starting at 0 |
| vector  | position | New spline point position in spline localspace              |
