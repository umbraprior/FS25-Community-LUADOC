### setVisibilityConditionVisibleShaderParameter

**Description**

> Set the shader parameter of the visibility condition

**Definition**

> setVisibilityConditionVisibleShaderParameter(entityId objectId, float shaderVisibilityParam)

**Arguments**

| entityId | objectId              | objectId                                                       |
|----------|-----------------------|----------------------------------------------------------------|
| float    | shaderVisibilityParam | shader parameter when condition is met (ie. object is visible) |
