### getXMLNumOfChildren

**Description**

> Returns the number of children of a given element

**Definition**

> getXMLNumOfChildren(entityId xmlId, string xmlElementPath)

**Arguments**

| entityId | xmlId          | xmlId                  |
|----------|----------------|------------------------|
| string   | xmlElementPath | Path to an xml element |

**Return Values**

| integer | numOfChildren |
|---------|---------------|
