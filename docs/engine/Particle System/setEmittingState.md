### setEmittingState

**Description**

> Set whether the particle system should emit new particles
> The particleSystemId can be retrieved by using getGeometry() on the shape/node

**Definition**

> setEmittingState(entityId particleSystemId, boolean state)

**Arguments**

| entityId | particleSystemId | particleSystemId |
|----------|------------------|------------------|
| boolean  | state            | state            |
