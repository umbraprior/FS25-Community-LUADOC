### setJointDrive

**Description**

> Set joint drive. Drives orientation if position drive or angular velocity if velocity drive.

**Definition**

> setJointDrive(integer jointIndex, boolean isLinear, boolean isPosition, float valueX, float valueY, float valueZ,
> float angle)

**Arguments**

| integer | jointIndex |
|---------|------------|
| boolean | isLinear   |
| boolean | isPosition |
| float   | valueX     |
| float   | valueY     |
| float   | valueZ     |
| float   | angle      |
