### getParticleSystemSpeed

**Description**

> Get particle system speed
> The particleSystemId can be retrieved by using getGeometry() on the shape/node

**Definition**

> getParticleSystemSpeed(entityId particleSystemId)

**Arguments**

| entityId | particleSystemId | particleSystemId |
|----------|------------------|------------------|

**Return Values**

| float | speed | speed |
