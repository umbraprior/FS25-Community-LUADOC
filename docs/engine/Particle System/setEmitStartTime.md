### setEmitStartTime

**Description**

> Set emitter starting time.
> The particleSystemId can be retrieved by using getGeometry() on the shape/node

**Definition**

> setEmitStartTime(entityId particleSystemId, float emitStartTime)

**Arguments**

| entityId | particleSystemId | particleSystemId |
|----------|------------------|------------------|
| float    | emitStartTime    | emitStartTime    |
