### getHasTrigger

**Description**

> Get if rigid body has 'trigger' flag set.
> Returns false if node does not exist or is not a rigid body.

**Definition**

> getHasTrigger(entityId shapeId)

**Arguments**

| entityId | shapeId |
|----------|---------|

**Return Values**

| boolean | hasTriggerFlag |
|---------|----------------|
