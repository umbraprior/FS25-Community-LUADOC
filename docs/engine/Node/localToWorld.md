### localToWorld

**Description**

> Local space to world space transformation

**Definition**

> localToWorld(entityId transformId, float lx, float ly, float lz)

**Arguments**

| entityId | transformId | transformId         |
|----------|-------------|---------------------|
| float    | lx          | local x translation |
| float    | ly          | local y translation |
| float    | lz          | local z translation |

**Return Values**

| float | wx | world x translation |
|-------|----|---------------------|
| float | wy | world y translation |
| float | wz | world z translation |
