### setEmitCountScale

**Description**

> Set particle system count scale
> The particleSystemId can be retrieved by using getGeometry() on the shape/node

**Definition**

> setEmitCountScale(entityId particleSystemId, float countScale)

**Arguments**

| entityId | particleSystemId | particleSystemId |
|----------|------------------|------------------|
| float    | countScale       | countScale       |
