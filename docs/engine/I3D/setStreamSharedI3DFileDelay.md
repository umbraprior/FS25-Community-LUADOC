### setStreamSharedI3DFileDelay

**Description**

> Set random delay parameters for shared I3D loading

**Definition**

> setStreamSharedI3DFileDelay(float minDelay, float maxDelay, float minDelayCached, float maxDelayCached)

**Arguments**

| float | minDelay       | min delay in seconds for loaded shared I3D |
|-------|----------------|--------------------------------------------|
| float | maxDelay       | max delay in seconds for loaded shared I3D |
| float | minDelayCached | min delay in seconds for cached shared I3D |
| float | maxDelayCached | max delay in seconds for cached shared I3D |
