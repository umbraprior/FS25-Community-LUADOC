> Euler angle to quaternion

> mathEulerToQuaternion(float x, float y, float z)

**Arguments**

| float | x | x |
|-------|---|---|
| float | y | y |
| float | z | z |

**Return Values**

| float | qx | qx |
|-------|----|----|
| float | qy | qy |
| float | qz | qz |
| float | qw | qw |
