### setOverlaySignedDistanceFieldWidth

**Description**

> Set overlay signed distance field width

**Definition**

> setOverlaySignedDistanceFieldWidth(entityId overlayId, float sdfWidth)

**Arguments**

| entityId | overlayId | overlayId                                                         |
|----------|-----------|-------------------------------------------------------------------|
| float    | sdfWidth  | range of the sdf in normalized color (0.5 full range, 0 disabled) |
