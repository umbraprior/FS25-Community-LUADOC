### setXMLInt

**Description**

> Set XML file integer value (in the range -2'147'483'648 - 2'147'483'647). If provided with a decimal number any
> decimals will be rounded in the saved value.

**Definition**

> setXMLInt(entityId xmlId, string nodePath, integer value)

**Arguments**

| entityId | xmlId    | xmlId                            |
|----------|----------|----------------------------------|
| string   | nodePath | path to XML element or attribute |
| integer  | value    | value                            |
