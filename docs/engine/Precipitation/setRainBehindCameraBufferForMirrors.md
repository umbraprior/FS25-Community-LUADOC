### setRainBehindCameraBufferForMirrors

**Description**

> Rain drops spawn a certain distance behind the camera as well in order to show up in mirrors, etc.
> This float value gives a percentage of the overall depth of the spawn box by which the spawn box is essentially
> shifted backwards.

**Definition**

> setRainBehindCameraBufferForMirrors(entityId precipitationEntity, float buffer)

**Arguments**

| entityId | precipitationEntity | precipitationEntity                                                                             |
|----------|---------------------|-------------------------------------------------------------------------------------------------|
| float    | buffer              | Percentage value by which the spawn box is shifted back to spawn rain behind the camera as well |
