### drawDebugArrowVector

**Description**

> Render an arrow. Only use for debug rendering

**Definition**

> drawDebugArrowVector(vector p, vector dir, vector tang, float r, float g, float b, boolean? solid)

**Arguments**

| vector   | p     |                                                                                                                             |
|----------|-------|-----------------------------------------------------------------------------------------------------------------------------|
| vector   | dir   |                                                                                                                             |
| vector   | tang  |                                                                                                                             |
| float    | r     |                                                                                                                             |
| float    | g     |                                                                                                                             |
| float    | b     |                                                                                                                             |
| boolean? | solid | [optional] true: depth is checked and point can be hidden behind other meshes; false: always rendered on top (default=true) |
