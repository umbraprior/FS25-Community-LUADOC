### addParticleSystemSimulationTime

**Description**

> Add particle system simulation time
> The particleSystemId can be retrieved by using getGeometry() on the shape/node

**Definition**

> addParticleSystemSimulationTime(entityId particleSystemId, float dt)

**Arguments**

| entityId | particleSystemId | particleSystemId |
|----------|------------------|------------------|
| float    | dt               | dt               |
