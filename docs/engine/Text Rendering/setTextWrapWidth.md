### setTextWrapWidth

**Description**

> Set text wrap width

**Definition**

> setTextWrapWidth(float wrapWidth, boolean? allowForcedWrap)

**Arguments**

| float    | wrapWidth       | wrapWidth                                                                       |
|----------|-----------------|---------------------------------------------------------------------------------|
| boolean? | allowForcedWrap | allow wrapping mid word when no separator is available [optional, default=true] |
