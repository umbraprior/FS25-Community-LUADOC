### setParticleSystemSpeed

**Description**

> Set particle system speed
> The particleSystemId can be retrieved by using getGeometry() on the shape/node

**Definition**

> setParticleSystemSpeed(entityId particleSystemId, float speed)

**Arguments**

| entityId | particleSystemId | particleSystemId |
|----------|------------------|------------------|
| float    | speed            | speed            |
