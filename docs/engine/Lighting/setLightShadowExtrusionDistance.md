### setLightShadowExtrusionDistance

**Description**

> Set shadow extrusion distance for light source. This is how far behind the user camera (from the perspective of the
> light source) shadows are still rendered so they can cast shadows into the camera region.

**Definition**

> setLightShadowExtrusionDistance(entityId lightId, float extrusion)

**Arguments**

| entityId | lightId   |          |
|----------|-----------|----------|
| float    | extrusion | distance |
