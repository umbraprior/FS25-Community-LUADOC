### worldRotationToLocal

**Description**

> World space to local space rotation transformation

**Definition**

> worldRotationToLocal(entityId transformId, float wrx, float wry, float wrz)

**Arguments**

| entityId | transformId | transformId      |
|----------|-------------|------------------|
| float    | wrx         | world rotation x |
| float    | wry         | world rotation y |
| float    | wrz         | world rotation z |

**Return Values**

| float | lrx | local rotation x |
|-------|-----|------------------|
| float | lry | local rotation y |
| float | lrz | local rotation z |
