**Description**

**Definition**

> addImpulse(entityId transformId, float impulseX, float impulseY, float impulseZ, float positionX, float positionY,
> float positionZ, boolean isPositionLocal)

**Arguments**

| entityId | transformId     | transformId     |
|----------|-----------------|-----------------|
| float    | impulseX        | impulseX        |
| float    | impulseY        | impulseY        |
| float    | impulseZ        | impulseZ        |
| float    | positionX       | positionX       |
| float    | positionY       | positionY       |
| float    | positionZ       | positionZ       |
| boolean  | isPositionLocal | isPositionLocal |
