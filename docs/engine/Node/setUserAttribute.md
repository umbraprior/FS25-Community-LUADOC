### setUserAttribute

**Description**

> Set user attribute value

**Definition**

> setUserAttribute(entityId objectId, string name, integer? type, any? value)

**Arguments**

| entityId | objectId | objectId                                                      |
|----------|----------|---------------------------------------------------------------|
| string   | name     | name                                                          |
| integer? | type     | Value of UserAttributeType enum [optional]                    |
| any?     | value    | value (type must match the type specified in type) [optional] |
