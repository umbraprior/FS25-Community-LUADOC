### utf8ToUnicode

**Description**

> Converts an utf8 string to unicode

**Definition**

> utf8ToUnicode(string utf8string)

**Arguments**

| string | utf8string | utf8string |
|--------|------------|------------|

**Return Values**

| integer | unicode | unicode |
|---------|---------|---------|
| integer | utf8Len | utf8Len |
