### setCenterOfMass

**Description**

> Set center of mass (only for dynamic physics objects)

**Definition**

> setCenterOfMass(entityId transformId, float x, float y, float z)

**Arguments**

| entityId | transformId | transformId |
|----------|-------------|-------------|
| float    | x           | x           |
| float    | y           | y           |
| float    | z           | z           |
