### mergeLightShadows

**Description**

**Definition**

> mergeLightShadows(entityId lightId, entityId lightId2, entityId lightId3, entityId lightId4, entityId lightId5,
> entityId lightId6, entityId lightId7, entityId lightId8, entityId lightId9, entityId lightId10)

| entityId | lightId   |                      |
|----------|-----------|----------------------|
| entityId | lightId2  |                      |
| entityId | lightId3  | optional light id 3  |
| entityId | lightId4  | optional light id 4  |
| entityId | lightId5  | optional light id 5  |
| entityId | lightId6  | optional light id 6  |
| entityId | lightId7  | optional light id 7  |
| entityId | lightId8  | optional light id 8  |
| entityId | lightId9  | optional light id 9  |
