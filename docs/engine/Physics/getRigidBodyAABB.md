### getRigidBodyAABB

**Description**

> Get rigid body AABB. Returns nil if transform is not added to physics (yet)

**Definition**

> getRigidBodyAABB(entityId transformId)

**Arguments**

| entityId | transformId | transformId |
|----------|-------------|-------------|

**Return Values**

| float | minX |
|-------|------|
| float | maxX |
| float | minY |
| float | maxY |
| float | minZ |
| float | maxZ |
