### getTerrainHeightAtWorldPos

**Description**

> Get terrain height at world pos

**Definition**

> getTerrainHeightAtWorldPos(entityId terrainId, float x, float y, float z)

**Arguments**

| entityId | terrainId | terrainId                  |
|----------|-----------|----------------------------|
| float    | x         | x                          |
| float    | y         | y (not relevant, can be 0) |
| float    | z         | z                          |

**Return Values**

| float | height | height |
|-------|--------|--------|
