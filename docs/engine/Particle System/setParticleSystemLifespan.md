### setParticleSystemLifespan

**Description**

> Set particle system life span.
> The particleSystemId can be retrieved by using getGeometry() on the shape/node

**Definition**

> setParticleSystemLifespan(entityId particleSystemId, float lifeSpan, boolean keepBlendTimes)

**Arguments**

| entityId | particleSystemId | particleSystemId |
|----------|------------------|------------------|
| float    | lifeSpan         | lifeSpan         |
| boolean  | keepBlendTimes   | keepBlendTimes   |
