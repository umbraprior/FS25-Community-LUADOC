### getSplitType

**Description**

> Get split type of shape
> Max supported value is 63.

**Definition**

> getSplitType(entityId shapeId)

**Arguments**

| entityId | shapeId | shapeId |
|----------|---------|---------|

**Return Values**

| integer | splitType | splitType, 0 if shape is not a mesh split shape |
|---------|-----------|-------------------------------------------------|
