### setDirectionVector

**Description**

> Set the direction of an object, the positive z-axis points towards the given direction. The y-axis lies in the
> direction-up-plane.
> Note that the scale of the node is reset to 1 1 1.

**Definition**

> setDirectionVector(entityId transformId, vector direction, vector up)

**Arguments**

| entityId | transformId |
|----------|-------------|
| vector   | direction   |
| vector   | up          |
