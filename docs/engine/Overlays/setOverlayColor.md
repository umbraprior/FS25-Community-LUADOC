### setOverlayColor

**Description**

> Set overlay color and alpha for whole overlay

**Definition**

> setOverlayColor(entityId overlayId, float red, float green, float blue, float alpha)

**Arguments**

| entityId | overlayId | overlayId |
|----------|-----------|-----------|
| float    | red       | red       |
| float    | green     | green     |
| float    | blue      | blue      |
| float    | alpha     | alpha     |
