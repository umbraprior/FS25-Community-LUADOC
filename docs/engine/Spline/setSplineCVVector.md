### setSplineCVVector

**Description**

> Set the position of a spline control point.

**Definition**

> setSplineCVVector(integer shapeId, integer index, vector position)

**Arguments**

| integer | shapeId  |                                                                |
|---------|----------|----------------------------------------------------------------|
| integer | index    | The index of the control point to be positioned, starting at 0 |
| vector  | position | New spline point position x in spline localspace               |
