### setWorldRotation

**Description**

> Set world rotation of a transform object

**Definition**

> setWorldRotation(entityId transformId, float x, float y, float z)

**Arguments**

| entityId | transformId | transformId |
|----------|-------------|-------------|
| float    | x           | x rotation  |
| float    | y           | y rotation  |
| float    | z           | z rotation  |
