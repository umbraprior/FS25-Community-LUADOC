### setTextClipArea

**Description**

> Sets clip area used for text rendering. Characters are only rendered within the clip area.
> Set to unit area (0,0,1,1) to disable.

**Definition**

> setTextClipArea(float clipMinX, float clipMinY, float clipMaxX, float clipMaxY)

**Arguments**

| float | clipMinX |
|-------|----------|
| float | clipMinY |
| float | clipMaxX |
| float | clipMaxY |
