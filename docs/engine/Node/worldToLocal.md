### worldToLocal

**Description**

> World space to local space translation transformation

**Definition**

> worldToLocal(entityId transformId, float wx, float wy, float wz)

**Arguments**

| entityId | transformId | transformId |
|----------|-------------|-------------|
| float    | wx          | world x     |
| float    | wy          | world y     |
| float    | wz          | world z     |

**Return Values**

| float | lx | local x |
|-------|----|---------|
| float | ly | local y |
| float | lz | local z |
