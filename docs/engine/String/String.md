### asciiToUtf8

**Description**

> Converts an ascii latin1 (ISO 88859-1) encoded string to an utf8 string

**Definition**

> asciiToUtf8(string asciiString)

**Arguments**

| string | asciiString | asciiString |
|--------|-------------|-------------|

**Return Values**

| string | utf8string | utf8string |
|--------|------------|------------|
