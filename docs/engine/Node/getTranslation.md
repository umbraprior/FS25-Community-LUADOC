### getTranslation

**Description**

> Get translation of a transform object in its local space (relative to parent node)

**Definition**

> getTranslation(entityId transformId)

**Arguments**

| entityId | transformId | transformId |
|----------|-------------|-------------|

**Return Values**

| float | x | x translation relative to parent node |
|-------|---|---------------------------------------|
| float | y | y translation relative to parent node |
| float | z | z translation relative to parent node |
