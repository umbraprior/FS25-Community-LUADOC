### getMaterialCustomParameter

**Description**

> Retrieves the custom parameter values of a material

**Definition**

> getMaterialCustomParameter(entityId materialId, string name)

**Arguments**

| entityId | materialId |                              |
|----------|------------|------------------------------|
| string   | name       | name of the custom parameter |

**Return Values**

| float | x | x |
|-------|---|---|
| float | y | y |
| float | z | z |
| float | w | w |
