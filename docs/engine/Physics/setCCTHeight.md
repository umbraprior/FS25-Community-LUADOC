### setCCTHeight

**Description**

> Set character height in meters of the cylinder part of the character capsule. Height will be reverted if the resizing
> leads to a collision.

**Definition**

> setCCTHeight(integer characterIndex, float height, integer collisionGroup, integer collisionMask)

**Arguments**

| integer | characterIndex | characterIndex |
|---------|----------------|----------------|
| float   | height         | height         |
| integer | collisionGroup | collisionGroup |
| integer | collisionMask  | collisionMask  |
