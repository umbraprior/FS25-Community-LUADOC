### addForce

**Description**

> Add force to object (only for dynamic physics objects)

**Definition**

> addForce(entityId transformId, float forceX, float forceY, float forceZ, float positionX, float positionY, float
> positionZ, boolean isPositionLocal)

**Arguments**

| entityId | transformId     | transformId     |
|----------|-----------------|-----------------|
| float    | forceX          | forceX          |
| float    | forceY          | forceY          |
| float    | forceZ          | forceZ          |
| float    | positionX       | positionX       |
| float    | positionY       | positionY       |
| float    | positionZ       | positionZ       |
| boolean  | isPositionLocal | isPositionLocal |
