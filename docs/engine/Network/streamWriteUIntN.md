### streamWriteUIntN

**Description**

> Stream write N-bit unsigned integer. 0<N<32.

**Definition**

> streamWriteUIntN(entityId streamId, integer value, integer numberofBits)

**Arguments**

| entityId | streamId     | streamId     |
|----------|--------------|--------------|
| integer  | value        | value        |
| integer  | numberofBits | numberofBits |
