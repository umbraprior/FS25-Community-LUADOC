### getXMLNumOfElements

**Description**

> Returns the number of equally named elements with the given path (the last element must not have an index, or the
> index must be 0)

**Definition**

> getXMLNumOfElements(entityId xmlId, string xmlElementPath)

**Arguments**

| entityId | xmlId          | xmlId                  |
|----------|----------------|------------------------|
| string   | xmlElementPath | Path to an xml element |

**Return Values**

| integer | numOfElements |
|---------|---------------|
